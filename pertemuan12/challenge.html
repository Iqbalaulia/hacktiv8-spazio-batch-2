My Favorite (Cats?)

Instruksi:

  1. Buat daftar dari barang / hewan favorit Anda
  2. Dalam daftar ini harus ada JS Object yang mengandung properties:
    • Bertipe String (ex: nama)
    • Bertipe Numeric (ex: umur)
    • Bertipe Boolean (ex: rumahan?)
    • Bertipe Array (ex: daftar makanan fav)
    • Gambar (boleh local file atau URL gambar)
  3. Tambahkan Bootstrap
  4. Tambahkan jQuery
  5. Tampilkan daftar diatas seperti pada: Practice 7
